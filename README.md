# SEC Filing Analyst Agent

Agentic SEC filing analyst — retrieves 10-K/10-Q filings via EDGAR, extracts structured data with a local model, and synthesizes analyst context via web search.

## Usage

```bash
uv run agent.py NVDA
uv run agent.py NVDA --model anthropic
uv run agent.py NVDA --json
```
